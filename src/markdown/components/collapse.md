## VlCollapse Attributes

参数   |   说明   |   类型   |   可选值   |   默认值
---- |:----:|:---- |:---- |:----:|
itemsData | 子项数据 | `Record<string, any>[]` | - | -
selectedArray | 默认开启标题 | `Array<string>` | - | -
onlyShowSingle | 是否展示唯一项 | boolean | true/false | false

## VlCollapseItem Attributes

参数   |   说明   |   类型   |   可选值   |   默认值
---- |:----:|:---- |:---- |:----:|
title  | 标题 |   String   | - |  -
isDisable | 是否禁用 |   Boolean   |   true/false   |  -
