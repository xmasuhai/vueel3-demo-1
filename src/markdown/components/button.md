## Attributes

参数   |   说明   |   类型   |   可选值   |   默认值
---- |:----:|:---- |:---- |:----:|
theme   |   主题样式   |   String   |  button / link / text  |   —
size   |  尺寸大小  |   String   |   big / medium / small  |   —
level   | 重要等级 |   String   |  normal / primary / danger   |   —
shape   |   形状   |   String   |   rect / capsule / circle   |   —
disabled   | 是否禁用 |   Boolean   |   —   |   —
isLoading   | 是否加载中 |   Boolean   |   —   |   —
