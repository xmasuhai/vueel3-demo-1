# Attributes

Icon图标将被渲染为span标签，width、height 、fill属性可以直接在组件标签上设置样式。

参数   |   说明   |   类型     |   可选值   |   默认值
---- |:----:|:---- |:---- |:----:|
iconName   |   icon 图标名   |   String   | 见下表 | -

## iconName 可选值与对应图标

<script setup>
import IconList from '@/markdown/components/iconList.md'
</script>

<IconList iconName="vite"></IconList>
